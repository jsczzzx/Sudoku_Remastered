## Features

- Developed by React Native
- Register & Login
- Easy Mode & Hard Mode
- Auto Correctness Checker
- Timer & Pause
- Local/Global Highscores
- Runs on iOS and Android

## Development Setup

```console
  yarn add
  yarn run
```
